### 7.7.7 sRGBA与sBGRA纹理的转换规则

从sRGB颜色空间到线性颜色空间的转换在对sRGB纹理进行采样的时候自动完成。


```metal
if (c <= 0.04045) 
    result = c / 12.92;
else 
    result = powr((c + 0.055) / 1.055, 2.4);
```
